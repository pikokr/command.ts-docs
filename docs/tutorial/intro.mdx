# 시작하기

## 템플릿 클론하기

```shell
npx degit https://github.com/pikokr/command.ts-v3-quickstart-template project-name
cd project-name
yarn
```

설치가 끝났다면 `config.json`을 수정해주세요.

```json title="config.json"
{
  "token": "봇 토큰",
  "slash": {
    "guild": "슬래시 커맨드를 적용할 서버 아이디"
  }
}
```

## 봇 실행하기

개발 환경에서 실행하기

```shell
yarn dev
```

프로덕션 환경에서 실행하기

```shell
yarn build # 타입스크립트 프로젝트 빌드
yarn prod # 프로젝트 실행
```
